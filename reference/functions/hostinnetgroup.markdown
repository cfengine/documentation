**Prototype**: `hostinnetgroup(netgroup)`

**Return type**: `class`

**Description**: True if the current host is in the named `netgroup`.

**Arguments**:

* `netgroup` : Netgroup name, *in the range* .\*

**Example**:

```cf3
    classes:

      "ingroup" expression => hostinnetgroup("my_net_group");
```
